   双向BFS学习的总结：
   	1、从出发地和目的地两个端点同时进行BFS搜索，同时用两个Set来标记已访问的元素
        2、每次通过交换队列实现从最小的队列开始下一层的搜索（同时交换彼此的必要信息）
	3、当bfs搜索到的新元素在另一个已访问队列中存在的时候，说明两个bfs路径相遇，即为可能的一条最短路径
